
  # Responsive Financial Adviser Website

  This is a code bundle for Responsive Financial Adviser Website.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Contact Form Email Setup

  The contact form uses EmailJS to send emails to the owner. To enable this feature:

  1. Install dependencies: `npm install` (includes @emailjs/browser)
  2. Follow the setup guide in [EMAILJS_SETUP.md](./EMAILJS_SETUP.md)
  3. Create a `.env` file with your EmailJS credentials
  4. Restart the development server

  For detailed instructions, see [EMAILJS_SETUP.md](./EMAILJS_SETUP.md).
